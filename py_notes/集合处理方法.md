# 集合处理方法

1. 集合间操作：交(&)、并(|)、差(-)、补(^)、比较(>=<)
| 操作函数或方法 | 描述                                                |
| S.add(x)       | 如果x不在集合S中，将增加到S                         |
| S.discard(x)   | 移除S中元素x，如果元素不在集合S中，不报错           |
| S.remove(x)    | 移除S中元素x，如果x不在集合S中，产生KeyError异常    |
| S.clear()      | 移除S中所有元素                                     |
| S.pop()        | 随机返回S的一个元素，更新S，若s为空产生KeyError异常 |
| S.copy()       | 返回集合S的一个副本                                 |
| len(S)         | 返回集合s的元素个数                                 |
| x in S         | 判断S中元素X，X在集合S中，返回True，否则返回False   |
| set(x)         | 将其他类型变量x转变为集合类型                       |

## 集合应用场景
