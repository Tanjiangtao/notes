面向过程，例如有张三打篮球，再做一个李四踢足球的程序？

面向对象，人的对象，人的运动的动作，运动的器械，这三个对象。
实例化一个张三的对象，对象有一个打篮球的动作，器械是篮球
实例化一个李四的对象，对象有一个踢足球的动做，器械是足球

面向对象能更好的在抽象的层面来分析问题，在程序实现跨度极大的赋予之前的代码这些是面向过程编程很难实现的
